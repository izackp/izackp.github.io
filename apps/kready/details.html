Product Description

The K-Ready app from the Orange County Library System (Orlando, FL) is a game that helps prepare preschoolers for kindergarten. It contains engaging and entertaining games that help preschoolers learn letters, numbers, shapes and colors. Bright custom graphics and encouraging voiceovers guide children through each level. Preschoolers will earn a reward each time a level of learning is completed! The full version of the app is available for free to all Orange County Library System resident cardholders in good standing
